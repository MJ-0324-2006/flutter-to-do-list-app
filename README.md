# To-Do List App ✅

A clean and simple Flutter To-Do app with 2 screens.

## Features
- View all tasks on main screen
- Add new tasks on second screen
- TextField input for task name
- State management using setState
- Clean and minimal UI

## Tech Stack
- Flutter & Dart
- setState for state management
